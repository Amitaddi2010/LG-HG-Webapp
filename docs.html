ESS Classification System

üìã Overview

The ESS (Endometrial Stromal Sarcoma) Classification System is an AI-powered web application that classifies histopathology images into Low Grade (LG) and High Grade (HG) categories using deep learning.

üß† Model Architecture

  • Base Model: EfficientNet-B0
  • Training: 5-Fold Cross-Validation
  • Classes: LG (Low Grade), HG (High Grade)
  • Input Size: 224x224 pixels
  • Preprocessing: ImageNet normalization

üìä Model Performance

The model was trained using 5-fold cross-validation on preprocessed histopathology patches with the following characteristics:

  • Data augmentation: rotation, flipping, color jittering
  • Class balancing through oversampling
  • Early stopping with patience mechanism
  • Weighted loss function for class imbalance

üîß Technical Details

Dependencies

  • Flask 2.3.3 - Web framework
  • PyTorch 2.0.1 - Deep learning
  • Torchvision 0.15.2 - Computer vision
  • Pillow 10.0.0 - Image processing
  • Flask-CORS 4.0.0 - Cross-origin requests

Image Processing Pipeline

  1. Convert to RGB format
  2. Resize to 224x224 pixels
  3. Normalize using ImageNet statistics
  4. Convert to tensor format
  5. Apply model inference
  6. Return softmax probabilities
